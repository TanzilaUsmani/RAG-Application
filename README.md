# 📚 RAG Book Question Answering System

An AI-powered Retrieval-Augmented Generation (RAG) application built using LangChain, ChromaDB, Hugging Face Embeddings, Mistral AI, and Streamlit.

## Features

- Upload PDF documents
- Automatic document chunking
- Embedding generation using Hugging Face
- Vector storage with ChromaDB
- Context-aware question answering using Mistral AI
- Interactive Streamlit interface

## Tech Stack

- Python
- LangChain
- Mistral AI
- Hugging Face Embeddings
- ChromaDB
- Streamlit

## Installation

```bash
git clone https://github.com/TanzilaUsmani/RAG-Application.git
cd RAG-Application
pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```env
MISTRAL_API_KEY=your_api_key_here
```
